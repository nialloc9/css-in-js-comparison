### Prerequisites

- install node: <a href="https://nodejs.org/en/download">https://nodejs.org/en/download</a>

## Install

```sh
    npm i -g yarn

    git clone git@github.com:nialloc9/css-in-js-poc.git

    cd css-in-js-poc

    npm install
```

### Run

```sh
    npm run compile
```

```sh
    npm run compare
```

### lint

```sh
    yarn lint
```
